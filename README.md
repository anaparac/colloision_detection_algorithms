# Detekcija_kolizija

Repozitorij za praktični dio diplomskog rada.

**Simple shapes**

Tipke za iscrtavanje objekata:
1 - kvadrat
2 - krug
3 - trokut
4 - romb

* **A** - box2d iscrtavanje oblika
* **B** - AABB

**CCD (Continous Collision Detection)**

Tipke za iscrtavanje objekata:

* **1** - meta
* **space** - ispaljivanje metka


* **A** - box2d iscrtavanje oblika
* **B** - AABB

**Nonconvex shapes**

Tipke za kretanje
* **left** - lijevo
* **right** - desno
* **space** - skok
* **up** - promjena kontrole nad igračima

* **A** - box2d iscrtavanje oblika
* **B** - AABB
